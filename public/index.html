  • Project Idea - Café

    As a student, finding available places on campus to study or do homework is a hassle. The only way to find a place with open seats is to walk around looking for places in the library or other places on campus. There is however, a potentially untapped resource, unused classrooms! The issue there, is twofold compared to the more conventional places. You still have to find open spaces, and you never know when the next class is going to interrupt you! Café (Classroom Availability Finding Éngine) is a webapp that uses the courseoff REST API to query for class times and use that to create a map of open classrooms for all times. From there, anyone can log on to see a timeline of any unused classrooms.
